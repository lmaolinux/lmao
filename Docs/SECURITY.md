# Security Policy
>[!Note]
> > We treat all security vulnerabilities, privacy, and system lock-down or hardening issues as security concerns. If an issue affects privacy, user freedom, or system security, please report it privately to protect everyone. Public disclosure can put all at risk before it is patched.
